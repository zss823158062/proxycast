## 架构说明

项目文档目录，包含技术规格、操作指南和文档站点配置。
使用 Nuxt Content 构建文档站点。

## 文件索引

- `content/` - 文档内容（Markdown）
- `images/` - 文档图片资源
- `TECH_SPEC.md` - 技术规格文档
- `LLM_FLOW_MONITOR_SPEC.md` - LLM 流量监控规格
- `ops.md` - 运维操作指南
- `app.config.ts` - Nuxt 应用配置
- `nuxt.config.ts` - Nuxt 框架配置
- `package.json` - 文档站点依赖
